Puzzle Cube Speed Timer

Puzzle Cube Speed Timer apps interface is built using UIKit. Utilizing UIButtons to start/stop the Timer. Upon stop of the timer, the time is recorded on to a UITableView which show your previous compleation times. Next to the UITableView displays data from your solve times. This data show fastest solve time, as well as average solve time. Generate random scramble algorithms, as well as select from multiple puzzle sizes. Data showed on the stats screen is dependent on the puzzle size selected.

Horror Movie Trivia

Horror Movie Trivia is a game I created using UIKit. Each new question enables the 30 second Timer to restart. If you get three incorrect answer you're dead; but guess as fast as you can to get the most points. Send a challenge to your friends using the UIActivityViewController share feature.

Indexed Flash Cards Maker

Indexed Flash Cards Maker was built using UIKit and a UICollectionView. Users can easily add new categories (decks) to seperate their study subjects. When reviewing each subject, questions are asked in a randomized order ensuring that the user learns the material, not just remember the answers in order they are asked.

Money Buddy Expense Tracker

Money Buddy Expense Tracker was built using UIKit. The income and expense lists are added into and displayed on UITableViews; while the expenses are also displayed as a pie chart using a Swift Package Manager package Charts. Navigate between different screens with the UITabBarController to display the income/expenses view, the calculator views, and a WKWebView.
